
# 🧠 Neural Text Classification

Этот проект демонстрирует, как можно использовать нейронные сети для задачи классификации текста. Реализация построена с использованием PyTorch и включает в себя полный пайплайн: от подготовки данных до обучения и оценки модели.

## 📝 Описание

В ноутбуке реализован базовый классификатор текстов на базе рекуррентной нейронной сети (LSTM). Цель — продемонстрировать основы обработки естественного языка (NLP) и построения моделей классификации с помощью PyTorch.

Пример задачи, под которую написан модуль:

- Определение тональности текста

## 🛠️ Технологии

- Python 3.10+
- PyTorch
- TorchText
- Scikit-learn
- Pandas / NumPy
- Matplotlib (для визуализации)

## ⚙️ Установка

1. Клонируйте репозиторий:

```bash
git clone https://github.com/Lakisor/IMDB_comments_classification
cd your-repo
```

2. Установите зависимости:

```bash
pip install -r requirements.txt
```

> При необходимости создайте `requirements.txt` командой:
> ```bash
> pip freeze > requirements.txt
> ```

## 🚀 Использование

Откройте Jupyter Notebook:

```bash
jupyter notebook
```

Загрузите файл `neural_text_classification.ipynb` и последовательно выполните ячейки.
